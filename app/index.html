hover-@animationClass # prefix modifier for any animation to enable hover only effects.
slide-@direction # add the left/right/up/down direction for sliding element
slide-far # modifier for slide-@direction slide length/duration longer
scale-@size # enables element to grow or shrink using up/down for @size
spin # enables element to spin
teeter # element will dip right and bounce back into position
bell # element will mimic lifting and shaking a bell
shake # cause element to shake right then left
flip # vertically 'flip' an element, example usage transitions
flip-card # horizontally 'flip' an element, example usage transitions
color-@color # change elements color to any of our colors by name
color-slow-@color # slowly change elements color to any of our colors by name
lose-background # slow fading of background to 0 opacity
